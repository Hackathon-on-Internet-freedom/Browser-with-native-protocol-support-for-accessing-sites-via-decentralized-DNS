# Browser-with-native-protocol-support-for-accessing-sites-via-decentralized-DNS
DNS записи основаны на аккаунтах из блокчейна VIZ, в которых хранятся необходимые записи об IP адресе сервера и пр. необходимая информация. Владелец аккаунта в любой момент может ее поменять. Обращение к сайтам идет по протоколу viz://account_name. При переходе по такой ссылке, идет запрос к нодам блокчейна, из которых берется необходимая информация для обращения к серверу сайта.
